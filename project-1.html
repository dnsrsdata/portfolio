Segmentação de clientes

Projeto para prever potenciais novos clientes de uma tabela com base em dados dos clientes atuais da empresa.

GitHub Link

Overview do projeto

Este é um projeto que faz parte do estágio virtual da KPMG, ele tem como objetivo prever potenciais novos clientes a partir de uma tabela, tendo dados dos clientes atuais da empresa como base para desenvolver a solução. Além disso, também foi feito uma Dashboard com o objetivo de mostrar os principais insights obtidos.

Dentre os resultados obtidos, destaco:
• Criação de um algoritmo que foi capaz de separar os clientes em dois grupos, onde o primeiro apresentou uma quantidade de compras 170% maior em relação ao segundo.
• O custo de produção dos itens do primeiro grupo também ficou menor, apresentando uma diferença de 10% em relação ao segundo.
