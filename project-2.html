Análise de dados do AirBnb

Projeto com o objetivo de obter insights dos dados do AirBnb do Rio de Janeiro.

GitHub Link

Project Overview

Este é um projeto que faz parte do Nanodegree Cientista de dados da Udacity. Nele eu analiso dados referentes ao AirBnb do Rio de Janeiro com o objetivo de obter insights. Apesar de não ser muito complexo, foi a primeira vez que tive contato com algumas bibliotecas usadas no desenvolvimento, como Folium e WordCloud, então tive que pesquisar um pouco sobre as mesmas.

Dentre as descobertas, estão:
• As áreas mais bem avaliadas do Rio de Janeiro.
• Quais termos eram os mais comuns de encontrar nas avaliações dos usuários.
• Disponibilidade de vagas para o ano de 2023.

Tecnologias utilizadas
